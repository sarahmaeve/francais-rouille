# French Institutions — Naming Reference

Current, historical, and commonly-confused names of French public
institutions and services that appear in our content. French
government bodies are renamed every few years; pedagogical material
dated later than the rename must use the current name.

**Authors:** before writing a dialogue set in a given year, check the
right column for anything your scene mentions. If in doubt, prefer
the most recent official name and note the predecessor parenthetically
only when the scene itself is deliberately recalling an earlier era.

**Reviewers:** this file is the authoritative check for institution
naming in any scene. Treat mismatches as factual errors on par with
getting the CAF acronym wrong.

## Employment and job-seeking

| Current name (use this) | Previous name | Change date |
|---|---|---|
| **France Travail** | Pôle emploi | 1 January 2024 |
| France Travail | ANPE + Assédic (merged into Pôle emploi) | 2008 |

*Mnemonic:* "Pôle emploi" appears in novels, films, and casual speech
for years after the rename — don't be misled. In a 2024-or-later
administrative scene, a préfecture agent or employer says *France
Travail*.

*Mnemonic:* titre-de-séjour dossiers are submitted exclusively on

## Commercial brand names vs administrative names

Some institutions have a commercial-sounding name that readers may
mistake for a brand:

- **Doctolib, Alan, Swile** — private commercial platforms, not public
  institutions. Use as any brand name would be used.
- **Ameli** — the online portal of the Assurance Maladie. Often
  said colloquially as if it's the institution itself ("J'ai un
  message sur Ameli"). OK in casual speech; say "Assurance Maladie"
  in formal writing.
- **Cerfa** — not an institution; a numbered form format. "Le Cerfa
  15186*03" means the form, not the agency issuing it.

## Reference sources

Service-public.fr is the official source of truth for administrative
procedures and current institution names. A few fiches we cite in
content:

- [F1618 — PACS](https://www.service-public.gouv.fr/particuliers/vosdroits/F1618)
  (conditions, procedure, and documents required when one or both
  partners is foreign)
- [F15763 — ANEF titre de séjour](https://www.service-public.gouv.fr/particuliers/vosdroits/F15763)
- [F14128 — Livret de famille](https://www.service-public.gouv.fr/particuliers/vosdroits/F14128)

When content introduces an administrative procedure, link the
relevant service-public fiche in a footnote or "Pour aller plus loin"
resource on the chapter page, so readers can verify the details.
