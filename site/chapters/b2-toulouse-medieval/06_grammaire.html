Grammaire B2 : Raconter l'histoire

Les outils grammaticaux de la visite guidée de Toulouse

Dans les dialogues de ce chapitre, Isabelle, notre guide, raconte l'histoire de Toulouse au Moyen Âge. Pour faire ce récit, elle utilise des constructions grammaticales qui vont au-delà du niveau B1. Voici les deux plus importantes.

Le plus-que-parfait

Le plus-que-parfait, c'est le temps qu'on utilise pour parler d'une action qui s'est passée avant une autre action dans le passé. On le forme avec l'imparfait de avoir ou être, plus le participe passé.

Par exemple, dans notre visite de Toulouse :

Dominique avait voyagé dans le sud de la France. Il avait compris que les méthodes de l'Église ne marchaient pas.

Dominique avait déjà voyagé et avait déjà compris — ces actions étaient terminées avant qu'il fonde l'ordre des Dominicains. C'est la différence essentielle : le plus-que-parfait place une action encore plus loin dans le passé.

Comparez

Plus-que-parfait (avant)Passé composé (après)
Les Cathares avaient continué à prêcher... ...alors le pape a lancé la croisade.
Dominique avait compris le problème... ...et il a fondé un nouvel ordre.
Le taureau avait traîné le corps... ...et il s'est arrêté à cet endroit.
Quand vous racontez une histoire au passé et que vous devez faire un retour en arrière — « mais avant ça, il avait déjà fait quelque chose » — c'est le moment d'utiliser le plus-que-parfait.

Les pronoms relatifs dont et où

Vous connaissez déjà qui (sujet) et que (objet direct). Voici maintenant deux autres pronoms relatifs, essentiels pour construire des phrases plus riches.

Dont — remplace de + quelque chose

Si le verbe ou l'expression utilise de, c'est dont qu'il faut utiliser.

C'est une histoire dont il faut se souvenir.

(= il faut se souvenir de cette histoire)

La ville dont les comtes régnaient sur un vaste territoire...

(= les comtes de cette ville régnaient sur un vaste territoire)

Le mouvement religieux dont nous avons parlé...

(= nous avons parlé de ce mouvement)

L'astuce : cherchez le de. Si vous dites « parler de », « se souvenir de », « avoir besoin de », « avoir peur de » — alors c'est dont.
Expression avec dePhrase avec dont
parler de l'Inquisition l'Inquisition dont nous avons parlé
avoir besoin de courage le courage dont ils avaient besoin
être fier de cette église l'église dont nous sommes fiers

Où — remplace un lieu ou un moment

Pour les lieux :

L'endroit où le taureau s'est arrêté est devenu sacré.

(= le taureau s'est arrêté à cet endroit)

La ville où les Dominicains se sont installés...

(= ils se sont installés dans cette ville)

Pour le temps :

L'époque où les Cathares prêchaient dans le sud...

(= les Cathares prêchaient à cette époque)

Le jour où Simon de Montfort a été tué...

(= il a été tué ce jour-là)
